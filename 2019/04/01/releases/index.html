Every PR as a release

01 Apr 2019

Last month we finally released sourmash 2.0 (yay!), after more than an year of alpha releases. Not committing to a final release was starting to get problematic, especially on tracking in which version something was added. Also, it was quite confusing to install from PyPI because pre-releases are not detected by default, forcing the users to run pip install --pre sourmash for the latest version.
