Science and the decentralized web

08 Aug 2018

Last week I attended the Decentralized Web Summit and while it was happening I was trying proof-of-concepts using IPFS and dat for some of our use cases in the lab. This is not a new plan (see the RECOMB 2017 poster 1, but it was always complicated to explore or expand it because both IPFS and dat Python modules usually lag behind their go or JavaScript counterparts 2. Some improvements both in tooling and technology maturity in the last few months closed the gap between plan and reality, so why not dedicate this week to see what is the current state?

  • Using IPFS to calculate signatures from GenBank, a demo taking advantage of the just released urlstore experimental feature.
  • Managing signature collections using dat, mirroring what other dApps are doing.
